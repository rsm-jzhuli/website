A simple visualization using Python & matplotlib

Visualization

This chart explores the relationship between car weight and fuel efficiency using a sample dataset similar to the classic mtcars.

Python Code


import matplotlib.pyplot as plt

# Sample data (car weight and MPG)
wt = [2.620, 2.875, 2.320, 3.215, 3.440, 3.460, 3.570, 3.190, 3.150, 3.440,
      3.440, 4.070, 3.730, 3.780, 5.250, 5.424, 5.345, 2.200, 1.615, 1.835,
      2.465, 3.520, 3.435, 3.840, 3.845, 1.935, 2.140, 1.513, 3.170, 2.770,
      3.570, 2.780]

mpg = [21.0, 21.0, 22.8, 21.4, 18.7, 18.1, 14.3, 24.4, 22.8, 19.2,
       17.8, 16.4, 17.3, 15.2, 10.4, 10.4, 14.7, 32.4, 30.4, 33.9,
       21.5, 15.5, 15.2, 13.3, 19.2, 27.3, 26.0, 30.4, 15.8, 19.7,
       15.0, 21.4]

plt.scatter(wt, mpg)
plt.xlabel('Weight')
plt.ylabel('Miles per Gallon (mpg)')
plt.title('Scatter Plot of Weight vs. MPG')
plt.grid(True)
plt.show()

Project Summary

In this project, I visualized the correlation between vehicle weight and fuel efficiency using Python and matplotlib. The scatter plot reveals a negative trend — as weight increases, MPG generally decreases. This quick visual demonstrates how accessible insights can be derived from simple data analysis.
